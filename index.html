Authentication, through proof of ownership.

Clave is a tool for developers to integrate NFT-based authentication and authroization into their projects. This allows for a whole new paradigm of ownership-based permissioning on the internet.


1. Login

User wallet connections are handled internally by Clave so that you don't need to maintain the code for each wallet.


2. Authentication

Clave checks user wallets for ownership of NFTs that you specify, then provides you with the results.


3. Authorization

Based on authentication from NFT ownership, users are given access permissions to whatever features you decide.
